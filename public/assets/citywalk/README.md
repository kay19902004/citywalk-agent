# CityWalk Asset Slots

Reserved paths for future production art.

- `hud/`: AR scan frame, markers, arrows, lock-on effects.
- `badges/`: explorer badges, rarity medals, profile emblems.
- `textures/`: map grid, paper/light glass textures, route overlays.
- `codex/`: collection thumbnails, locked clues, mystery placeholders.

Current UI uses CSS/SVG-like fallbacks, so missing art assets will not break pages.

Processed PNG assets from `qingkesong` have been normalized into these folders with product-facing names. HUD, badge, route-panel, and codex-book PNGs use real alpha where needed; street/map/report backgrounds remain normal RGB images.
